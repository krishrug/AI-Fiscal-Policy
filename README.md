# Thailand Expenditure 2016-2026 AI Analysis

## Project Description
This project focuses on analyzing the fiscal policies and expenditure patterns of Thailand from the years 2016 to 2026. Using AI techniques, we aim to forecast future expenditures and assess the impact of various fiscal policies implemented during this period.

## Objectives
1. **Data Collection:** Gather relevant data on Thailand's expenditures from 2016 to 2026.
2. **Data Analysis:** Utilize machine learning algorithms to analyze the data and identify trends.
3. **Visualization:** Create interactive visualizations to represent our findings.
4. **Forecasting:** Develop models to predict future expenditures based on historical data.
5. **Policy Assessment:** Evaluate the effectiveness of fiscal policies through quantitative measures.

## Setup Instructions
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/krishrug/AI-Fiscal-Policy.git
   cd AI-Fiscal-Policy
   ```
2. **Install Dependencies:**
   Ensure you have Python and pip installed, then run:
   ```bash
   pip install -r requirements.txt
   ```
3. **Access the Data:**
   Data files are located in the `data` directory. Ensure to replace any placeholder data with actual datasets.
4. **Run the Analysis:**
   Execute the analysis scripts provided in the `analysis` directory to start the forecasting model.

5. **View Results:**
   Visualizations and results will be available in the `results` folder after running the analysis scripts.
